# TempControlledServo

A program that controls a servo using a sensor that monitors temperature.
COMPONENTS:
Arduino Due Board
DHT11 Temperature and Humidity Sensor
LCD 1602 Display
4 LEDS
4 330ohm Resistor
Servo

*schematic is attached with the code.


test code for each component is also provided.

libraries required:
<dht.h>
<Servo.h>

<LiquidCrystal.h>
